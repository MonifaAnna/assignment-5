Important FAQ

What is the Document Object Model(DOM) ?

The Document Object Model (DOM) is a programming interface that represents a web page's structure as a tree of objects.
It allows developers to interact with and manipulate HTML, CSS, and content dynamically using JavaScript.

How do you select an element from the DOM ?

To select an element from the DOM, you can use JavaScript methods like:

  • document.getElementById('id') – Selects an element by its ID.
  • document.getElementsByClassName('class') – Selects all elements with a specific class.
  • document.getElementsByTagName('tag') – Selects all elements with a specific tag name.
  • document.querySelector('selector') – Selects the first element that matches a CSS selector.
  • document.querySelectorAll('selector') – Selects all elements that match a CSS selector.
What is the event delegation in the context of the DOM ?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of
adding multiple listeners to child elements. The event listener takes advantage of event bubbling, allowing it to detect
and handle events triggered by any of its child elements.
This method improves performance and simplifies code, especially when working with dynamic elements.

How do you manipulate an elements attributes and style using DOM?

To manipulate an element's attributes and styles using the DOM:

  • Attributes: Use setAttribute() to set an attribute and getAttribute() to retrieve it.
  • Example:element.setAttribute('src', 'image.jpg');
  • Styles: Access the style property to modify CSS styles directly.
  • Example:element.style.color = 'red'; element.style.fontSize = '20px';
